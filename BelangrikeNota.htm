Belangrike nota vir Windows XP SP2 gebruikers

Weens veranderinge in die metode hoe Microsoft Internet Explorer web bladsye hanteer mag die inhoud van hierdie bladsye nie altyd vertoon soos dit bedoel was nie. Dit is spesifiek van toepassing met Windows XP SP2.

Wanneer die boodskap soos bo gewys word kliek enige plek op die geel blok en kies. Die volgende keuselys sal opkom:

Kies 'Allow Blocked Content. Die volgende boodskap sal dan opkom:

Kies 'Yes'.

Ongelukkig is dit nou deel van hoe Internet Explorer werk en kan ons niks daaraan doen nie.
