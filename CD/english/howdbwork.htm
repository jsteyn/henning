HOW TO USE THE DATABASE

The database allows users several methods to search for the particulars of a person:

Surnames of persons If the surname of a person related by marriage is known, you can start here
Birthplace of persons If the birthplace is known, you can start here. A person can only be reached, if this information is recorded.
Birth year of persons A person can only be reached if this information is recorded.
First names of men You can start here, if the first names of a man is known.
First names of women You can start here, if the first names of the woman is known.
Progenitor If the genealogical number of the person you are looking for, is available, you can start navigating from the progenitor.

If a name is underlined when you move the cursor over it, it indicates that you can click on this name and navigate to the record of that person.
